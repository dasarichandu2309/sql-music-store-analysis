# 📄 SQL Queries

This folder contains all SQL queries used in the project.

## 📌 Description
The queries are written to analyze the music store database and extract business insights.

## 📊 Includes
- Customer analysis queries
- Sales and revenue queries
- Genre and artist analysis
- Advanced SQL (joins, subqueries, aggregations)

## 🚀 Usage
Run these queries in your SQL environment (MySQL/PostgreSQL) after importing the dataset.
